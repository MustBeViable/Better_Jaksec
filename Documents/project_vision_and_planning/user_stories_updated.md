# User Stories

* As a user I want to log in securely so I can access the system with the correct permissions. **(5)**
* As a user I want to change my password so I can secure my account. **(1)**
* As a user I want to change the system language so I can use the application in my preferred language. **(1)**
* As a user I want to see all courses I have access to so I know what I am enrolled in. **(2)**
* As a teacher I want to create and manage my own courses so I can organize teaching. **(3)**
* As a teacher I want to add and remove students from my courses so I can manage enrollment. **(2)**
* As a teacher I want to start and end attendance tracking for a class so students can mark their attendance during the session. **(4)**
* As a teacher I want to mark student attendance so attendance records are recorded correctly. **(1)**
* As a student I want to mark my own attendance so my presence is recorded. **(2)**
* As a student I want to view my attendance per course so I can track my absences. **(1)**
* As a student I want to submit a reason for absence so it can be stored in the system. **(1)**
